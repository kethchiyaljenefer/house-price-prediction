# 🏠 House Price Prediction

A machine learning project that predicts house prices using Linear Regression based on various features like location, size, bedrooms, etc.

# 📊 Dataset
- Source: Kaggle House Price Dataset
- Features: Area, Bedrooms, Bathrooms, Stories, Parking, etc.
- Target: Price

# 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Model:** Linear Regression
